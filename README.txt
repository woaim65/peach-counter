========================================
           桃数统计 使用说明
========================================

不需要懂电脑，照着下面步骤做就行。


【第一步】装 Python
----------------------------------------

1. 打开这个网址：
   https://www.python.org/downloads/

2. 点那个黄色的 "Download Python 3.x.x" 按钮

3. 下载完双击安装包

4. 重要！！！安装界面最下面有个框：
   [ ] Add Python to PATH
   把它打勾！！！（不勾的话后面全废）

5. 点 "Install Now"，等它装完


【第二步】启动程序
----------------------------------------

1. 把整个文件夹解压到桌面

2. 双击里面的 run.bat

3. 第一次启动会黑框闪一会儿（在装依赖），等两三分钟

4. 装完自动弹出程序窗口，就可以用了


【第三步】怎么用
----------------------------------------

----- 单张图片 -----
点 "单张选图" → 选一张照片 → 弹窗告诉你几个

----- 批量处理（推荐）-----
点 "批量选文件夹" → 选存照片的文件夹
→ 程序自动扫里面所有图片，一张张处理
→ 表格显示每张的个数
→ 底下显示总落果数

----- 图片要求 -----
- 拍照时手机正对着地面拍，不要斜着
- 桃子掉在地上的那种场景最准
- 地上如果有套桃子的纸袋，先踢开再拍
- 晴天光线好效果最好，阴天也行


【结果图】
----------------------------------------

每张原图旁边会自动生成 xxx_result.jpg
上面框出了每个桃子并编了号，方便你核对


【选什么模式】
----------------------------------------
- 自动（推荐）：啥都不用管，程序自己判断
- 熟桃：桃子颜色偏黄偏红的时候选
- 青桃：桃子还青、没熟透的时候选
  拿不准就选"自动"


【常见问题】
----------------------------------------

Q: 双击 run.bat 闪一下就没了？
A: Python 没装好。重新按第一步装，记得勾 "Add Python to PATH"

Q: 提示 "python 不是内部或外部命令"？
A: 同上，没加 PATH。重装 Python，勾那个框

Q: 处理结果不准？
A: 试试换一个模式（熟桃/青桃/自动），或者拍照时保持光线好一点

Q: 能处理多少张图？
A: 文件夹里放几百张都行，就是跑得慢一点
